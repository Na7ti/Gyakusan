# Gyakusan - 逆算学習計画アプリ 🎯

## 概要 (Overview)
**Gyakusan**は、試験日と目標から「今やるべきこと」を逆算して提示する学習管理アプリケーションです。
Gemini AIを活用して最適な学習ロードマップを自動生成し、Googleカレンダーと連携してスケジュール管理を自動化します。

## 主な機能 (Features)

### 1. 📅 試験管理 (Exam Management)
- 試験日、開始日、目標ページ数を設定するだけで、独自の「逆算スケジュール」を作成。
- 複数の試験を登録し、ダッシュボードで一元管理。

### 2. 🤖 AI学習ロードマップ (AI Roadmap)
- 「AI学習計画を生成」ボタンをクリックするだけで、Gemini APIが試験内容に合わせた具体的な学習トピックを提案。
- 重要なトピックのリストアップと、学習の優先順位付けをサポート。

### 3. 🔄 Googleカレンダー連携 (Calendar Sync)
- 生成された学習スケジュール（「第X日: Yページまで」）をGoogleカレンダーに自動でイベントとして登録。
- 出願期間や支払期限などの重要イベントも見逃しません。

### 4. 📊 ダッシュボード (Dashboard)
- **残り日数**: 試験までのカウントダウンを表示。
- **今日のタスク**: その日にやるべき具体的なノルマ（例: 「P.50まで進む」）を明確に表示。
- **進捗バー**: 期間に対する現在の位置を視覚化。

## 使い方 (Usage)

1.  **ログイン**: アプリにアクセスし、Googleアカウントでログインします（開発環境ではMock Loginも利用可）。
2.  **試験作成**:
    - 「試験を作成」ボタンを押下。
    - 試験名、実施日、テキストの総ページ数を入力して保存。
3.  **AI計画生成**:
    - 作成した試験の詳細ページへ移動。
    - 「AI学習計画を生成」ボタンをクリックし、ロードマップが生成されるのを待ちます。
4.  **日々の学習**:
    - トップページ（ダッシュボード）で「今日のタスク」を確認。
    - 完了したらチェックを入れ、カレンダーの予定通りに進めます。

## 技術スタック (Tech Stack)
- **Frontend/Backend**: [Deno](https://deno.land/) + [Fresh](https://fresh.deno.dev/)
- **Database**: PostgreSQL (Dockerized)
- **AI**: Google Gemini API
- **Infrastructure**: Docker Compose

## 開発・テスト (Development)

### ローカル実行
```bash
docker-compose up --build
```
`http://localhost:8000` でアクセス可能。

### 自動テスト実行
統合テストスクリプト（認証・作成・AI・削除の一連フロー）を実行して動作確認ができます。

```bash
# Dockerコンテナが起動している状態で実行
bash scripts/verify_curl.sh
```
