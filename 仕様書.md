資格試験日程管理アプリ 仕様書 (完全版)
1. コンセプト

「合格から逆算する、挫折しない伴走者」 単なる予定表ではなく、試験日から逆算して「今日、何をすべきか」を自動で提示し、Googleエコシステムと連携することで、学習を日常のルーティンに組み込むためのプラットフォーム。
2. ターゲットユーザー

    計画が苦手な受験者: どこから手をつければいいか分からず、直前で焦ってしまう人。

    多忙な社会人・学生: 隙間時間を活用し、Googleカレンダーで私生活と勉強を一括管理したい人。

    モチベーション維持が課題の人: 自分の頑張りを数値やグラフで実感し、継続の糧にしたい人。

3. 主要機能 (MVP)
3.1 認証・外部連携

    Google OAuth 2.0 ログイン: 面倒な登録を省き、セキュアに利用開始。

    Google Calendar API 同期: 学習計画を自分のカレンダーに自動反映。

3.2 スケジュールエンジン

    逆算プランナー: 試験日と目標学習時間（または教材の総ページ数）を入力すると、試験日までの日割りノルマを自動算出。

    重要日程アラート: 願書受付開始日、締切日、受験料支払い期限をカレンダーに自動登録。

3.3 進捗管理・ダッシュボード

    本日のタスク: 起動時に「今日やるべきページ数/時間」を明確に提示。

    進捗トラッカー: 学習完了をチェックすると、DBに記録され進捗率が更新。

4. 拡張機能 (将来的なアップデート)

    AI学習ロードマップ (Gemini API): 資格名を入れると、効率的な学習順序や頻出分野をAIがアドバイス。

    忘却曲線復習システム: 過去の学習記録から、適切なタイミングで「復習タスク」を自動挿入。

    Gmail自動スキャン: 受験申し込みメールから試験日や会場情報を自動で抽出・登録。

5. UI/UXイメージ

    Dashboard: 試験までのカウントダウンタイマーを最上部に配置し、適度な緊張感を演出。

    Visual Feedback: 学習完了ボタンを押すと、グラフが伸びるアニメーションや「積み上げ」を視覚化したデザイン。

    Minimal Input: 入力項目を極限まで減らし、勉強の邪魔をしないシンプルな操作感。

6. 技術スタック (確定)
レイヤー	技術選定
Container	Docker / Docker Compose
Runtime	Deno (Fresh or Hono)
Database	PostgreSQL (Local Container)
Version Control	Git
API	Google Calendar API / Google OAuth
7. データモデル案

    users: ユーザー基本情報、Google連携トークン

    exams: 試験名、試験日、目標数値（時間/ページ）、現在の進捗

    tasks: 日ごとのノルマ、完了フラグ、GoogleカレンダーイベントID
